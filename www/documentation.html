As a package, each component of adegenet is documented in help pages. As a starting point, one can type ?adegenet in R to get an overview of package.

The following other ressources are available:
- the adegenet manual (pdf), which compiles all the documentation of the package.
- the adegenet tutorial (pdf), which goes through the main functionalities of the package. Regularly updated.
- files required for some parts of the tutorial
       > AFLP.txt: a toy dataset reproducing AFLP data
       > haplo.txt: a toy dataset reproducing haploid data
       > triplo.txt: a toy dataset reproducing triploid data
- worksOnyInDiploids.txt: the list of functions working only for diploid genotypes
- a sPCA tutorial (pdf), showing how to use the spatial Principal Component Analysis (Jombart et al. 2008)
- globaldiv summer school 2009 - talk - practical course
