As a package, each component of adegenet is documented in help pages. As a starting point, one can type ?adegenet in R to get an overview of package.

The following other ressources are available:
- the adegenet manual (pdf), which compiles all the documentation of the package.
- the adegenet tutorial (pdf), which goes through the main functionalities of the package.
- files required for some parts of the tutorial
       > AFLP.txt: a toy dataset reproducing AFLP data

Related publications:
- a paper presenting the sPCA, global and local tests:
    Jombart T, Devillard S, Dufour AB, Pontier D (2008) Revealing cryptic spatial patterns in genetic variability by a new multivariate method.  Heredity 101: 92-103. doi: 10.1038/hdy.2008.34 [link on the journal's website - free abstract - email me for a pdf]
- an application note presenting the adegenet package:
    Jombart T. (2008) adegenet: a R package for the multivariate analysis of genetic markers. Bioinformatics 24: 1403-1405. doi: 10.1093/bioinformatics/btn129 [link to a free pdf]
