As a package, each component of adegenet is documented in help pages. As a starting point, one can type ?adegenet in R to get an overview of package. Other sources of information distributed with the package include a manual and vignettes. This website also provides other resources such as lectures and practicals.

To search quickly through these resources, use the search engine.



Manuals
- the adegenet manual (pdf), which compiles all the manpages of the package.


Tutorials vignettes
- adegenet-basics: introduction to the package; accessible from R by typing vignette("adegenet-basics").
- adegenet-spca: spatial multivariate analysis using adegenet; includes an introduction to spatial PCA (sPCA); accessible from R by typing vignette("adegenet-spca").
- adegenet-dapc: analysis of the genetic structuring of populations; includes an introduction to Discriminant Analysis of Principal Components (DAPC); accessible from R by typing vignette("adegenet-dapc").
- adegenet-genomics: introduction to the handling of genome-wide SNP data using adegenet; accessible from R by typing vignette("adegenet-genomics").
