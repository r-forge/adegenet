adegenet is a  package dedicated to the handling of molecular marker data for multivariate analysis. This package is related to ade4, a R package for multivariate analysis, graphics, phylogeny and spatial analysis. It is currently hosted by R-Forge. adegenet is described in the following application note:
Jombart T. (2008) adegenet: a R package for the multivariate analysis of genetic markers. Bioinformatics. doi: 10.1093/bioinformatics/btn129 [link to a free pdf]

Adegenet starting, Monmonier  algorithm, analyses using ade4, ...
Main features of adegenet are:

- definition of new object classes to facilitate multivariate analysis
- data import from GENETIX, STRUCTURE, Genepop, Fstat, Easypop, or any dataframe of genotypes
- data export to the R packages genetics, hierfstat, LDheatmap
- HWE and G-statistic tests, F statistics implemented for adegenet objects
- computation of genetic distances
- basic data information and advanced functions for data handling
- methods for spatial genetics: Monmonier algorithm, spatial PCA, tests for global and local structuring
